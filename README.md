## Overview

<p>This repository demonstrates a gRPC client microservice that communicates with the Product Service to retrieve product details required for order listing.</p>

<p>This service consumes gRPC APIs but does not expose any gRPC endpoints.</p>

<p>For server side implementation you can checkout to this repository https://github.com/Progesh/laravel-grpc-server</p>
